# Окружение для QA-автоматизации

Инструкции по созданию виртуального окружения и установке зависимостей (macOS):

1) Запустить подготовленный скрипт (создаёт `.venv` и устанавливает зависимости):

```bash
cd $(dirname "$0")
chmod +x setup_env.sh
./setup_env.sh
```

2) Активировать виртуальное окружение вручную (если нужно):

```bash
source .venv/bin/activate
```

3) Запустить тесты (пример с pytest):

```bash
pytest -q
```

Если вы хотите использовать другую версию Python, замените `python3` на путь к нужному интерпретатору.
